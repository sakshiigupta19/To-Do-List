This project is a simple To-Do List web application built with HTML, CSS, and JavaScript. It allows users to add, delete, and mark tasks as complete. 
The project demonstrates basic web development skills and is a great addition to your GitHub portfolio.

Features
Add new tasks
Delete tasks
Mark tasks as complete

index.html: The main structure of the application.
styles.css: The styling for the application.
script.js: The functionality of the application.
